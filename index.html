Introduction to CSS

Computer System Servicing (CSS) involves installing, maintaining, and troubleshooting computer systems and networks. It ensures smooth operations of devices and connectivity.

Setting Up a Wi-Fi Network

Wi-Fi networks allow devices to connect wirelessly to the internet. Setting up involves acquiring a router, configuring security settings, and ensuring proper connectivity.

Wi-Fi operates on two primary frequency bands: 2.4 GHz and 5 GHz, affecting range and speed. WPA2 and WPA3 encryptions secure Wi-Fi connections against threats.

Network Documentation

Network documentation records all network-related details, including IP allocations, hardware inventory, and configurations. Proper documentation aids in troubleshooting and network management.

Key components include network topology, hardware, software, and IP address allocations. It is essential for network recovery, troubleshooting, and scalability planning.

Types of Network Topologies

  • Star Topology: All devices connect to a central hub.
  • Ring Topology: Devices form a ring, each connected to two others.
  • Bus Topology: All devices share a single communication line.
  • Mesh Topology: Each device connects to multiple others for redundancy.
  • Wireless Mesh: Uses multiple wireless access points to extend coverage.

Network Security

Securing a network involves setting up strong passwords, using WPA2/WPA3 encryption, and monitoring traffic to prevent unauthorized access.

Encryption protects data from being intercepted. Public Wi-Fi networks are more vulnerable, whereas private networks use stronger encryption to prevent unauthorized access.

Best Practices for Network Management

Maintaining a stable network requires proper documentation, routine security updates, and monitoring for issues. Regular evaluations ensure efficiency and prevent failures.

Key steps include:

  • Establishing a documentation policy to ensure up-to-date network records.
  • Creating a network topology diagram to visualize infrastructure.
  • Labeling and inventorying all network devices.
  • Regularly evaluating documentation for completeness and accuracy.
